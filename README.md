### 개발 스택

- Frontend : ReactNative
- Backend : Node.js + NestJS

이유 : 프론트엔드는 expo기능을 활용해 ios/android를 동시에 빠르게 개발하기 위함. Node.js는 동일한 typescript환경에서 개발하기 위해 선택. 풀스택 개발자로써 API를 빠르게 개발하고 MVP를 출시하기 위함. Spring도 선택지 중 있었지만, JAVA/Kotlin기반이고, 무엇보다도 귀찮음

### MVP 기능 명세서

- [ ]  로그인 / 회원가입
- [ ]  방 만들기 - 방의 코드를 통해 입장하는 구조
- [ ]  방에 사진 올리기
- [ ]  갓생 인정해주기 - 포인트 기반 (인정은 선착순으로 한명만 해주면됨)

### 서비스 핵심 문장

‘인정이요’는 오늘 하루 열심히 산 순간에 대한 기록이며, 열심히 살고자 하는 사람들이 만나 서로를 인정해주며 앞으로 나아가는 앱이다.
